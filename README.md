# CRT Ambar — Classless CSS Demo

Demo de un **CSS classless** con estética retro de terminal CRT en tonos ámbar. El estilo se aplica solo con etiquetas HTML semánticas (`header`, `nav`, `main`, `article`, `aside`, etc.) sin necesidad de clases. Incluye efectos visuales tipo CRT: scanlines, flicker y ruido, aplicados vía JavaScript.
