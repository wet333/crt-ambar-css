> NOMBRES QUE SIGNIFIQUEN ALGO

VARIABLES Y FUNCIONES DEBEN DECIR QUÉ HACEN O QUÉ CONTIENEN. EVITAR ABREVIATURAS OSCURAS; PREFERIR NOMBRES LARGOS Y CLAROS A CÓDIGO CRÍPTICO.

> FUNCIONES PEQUEÑAS

UNA FUNCIÓN, UNA RESPONSABILIDAD. SI PUEDES EXTRAER UN BLOQUE CON UN NOMBRE DESCRIPTIVO, HAZLO. REDUCE COMPLEJIDAD CÍCLOMÁTICA Y FACILITA PRUEBAS.

> COMENTARIOS ÚTILES

EL MEJOR COMENTARIO ES CÓDIGO QUE NO NECESITA EXPLICACIÓN. COMENTAR EL "POR QUÉ" CUANDO EL "QUÉ" NO SEA OBVIO; EVITAR COMENTARIOS REDUNDANTES O DESACTUALIZADOS.
